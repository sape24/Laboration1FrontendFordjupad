# Laboration 1
En boksamling byggd i React + TypeScript som visar olika böcker med diverse information om respektive bok samt om de är lästa.

## Om webbsidan
Webbsidan tar data med böcker och visar dem i en lista fylld med BookCard komponenter, en header komponent med en banner samt en footer komponent finns. Även komponentspecifik CSS är implementerad.

## Teknik
- React
- TypeScript
- Vite
- Render
